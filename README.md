# Discrete-Math

- Little Discrete Math Programs.

---

Command to Compile each Program:
- `gcc OptyLib.c {0}.c {0}.h -lm -o {1}.out`

Arguments:
- `{0}`: Program Pathname
- `{1}`: Program Name
